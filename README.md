# Super Market Sales Analysis

## Description

This Power BI project presents a sales dashboard built using a supermarket sales dataset. The dashboard is designed to help users understand sales performance, identify business trends, and compare results across products, customer segments, and regions.

## Features

- Key performance indicators (KPIs) such as total sales, total orders, and average sales
- Sales by category analysis to compare major product groups
- Monthly sales trend chart to track changes over time
- Segment analysis to understand customer group performance
- Region analysis to compare sales across different areas
- Top products view to identify the best-performing items

## Tools Used

- Power BI
- Supermarket sales dataset (`Data/train.csv`)

## Dataset Information

The project uses a supermarket sales dataset containing order-level sales records. The dataset includes useful fields such as:

- Order Date
- Segment
- Region
- Category
- Sub-Category
- Product Name
- Sales

## Dashboard Overview

The dashboard helps users gain insights such as:

- Overall sales performance at a glance
- Which product categories contribute the most to sales
- How sales change month by month
- Which customer segments generate higher sales
- Which regions perform better than others
- Which products are the top contributors to revenue

## Tableau Dashboard

This repository also includes a Tableau-based sales dashboard analysis designed to present business performance through interactive visual storytelling. The Tableau workbook focuses on helping users evaluate overall sales performance, compare category and segment contributions, identify high-performing products, and explore geographic and time-based patterns in the data.

### Tableau Project Overview

The Tableau Sales Dashboard Analysis project provides a clean and interactive view of supermarket sales data for business reporting and exploratory analysis. It is intended to support quick decision-making by combining summary KPIs with deeper views across products, customer segments, locations, and sales trends.

### Tools Used

- Tableau

### Key Features

- KPIs to highlight core sales performance metrics at a glance
- Category analysis to compare the contribution of major product groups
- Segment analysis to evaluate performance across customer segments
- Top products view to identify the highest-performing items
- Map visualization to explore sales distribution by location
- Trend analysis to track performance over time
- Heatmap view to quickly identify high- and low-performing areas

### Tableau Workbook

The Tableau packaged workbook included in this repository is `Sales Data Analysis (Tableau).twbx`.

## How to Use

1. Open Power BI Desktop on your system.
2. Go to the project folder and locate the file `Sales Data Analysis (DV).pbix`.
3. Open the `.pbix` file in Power BI Desktop.
4. If required, refresh the data connection.
5. Explore the dashboard pages, filters, and visuals to view insights.

## Conclusion

This project demonstrates how Power BI can be used to turn supermarket sales data into meaningful business insights. It is a simple and effective student project for analyzing sales trends, customer segments, product performance, and regional sales patterns.
